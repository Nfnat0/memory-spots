## English Metadata

### Promotional Text

Turn familiar place photos into memory routes, then review visual notes by theme.

### Description

Memory Spots is a lightweight app for practicing the method of loci with photos of real places.

Create a route from familiar places such as your room, desk, commute, or favorite park. Add locations to photos, see them on a Memory Map, place visual notes directly on each photo, and switch themes to reuse the same route for different topics.

Use Memory Spots for exam prep, shopping lists, language practice, presentation cues, or anything that becomes easier to remember when it is attached to a place.

Features:

- Create ordered albums from place photos
- Add photos from the library or camera
- Show geotagged photos on a Memory Map
- Add or change a photo location later
- Organize notes by theme
- Place sticky text, image notes, icons, number labels, and arrows on photos
- Zoom and pan while editing a photo
- Review notes in route order by album and theme
- Keep data local on your device

Memory Spots is intentionally small and tactile. It helps you build a walkable path in your head, then attach what you want to remember to that path.

### Review Notes

This app stores user-created memory albums, notes, photos, optional photo coordinates, and app preferences locally on device. It does not require an account, does not include paid content, does not use analytics or advertising SDKs, and does not transmit user data to an external server.

The app asks for Photos access to add user-selected photos and read location metadata when available, Camera access to capture place photos, and Location access to show the user's current position on the map and attach coordinates to captured photos.
