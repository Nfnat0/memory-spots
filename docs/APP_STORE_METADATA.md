## English Metadata

### Promotional Text

Turn familiar place photos into memory routes, then review notes by theme.

### Description

Mind Palace is a simple app for practicing the method of loci with photos of real places.

Create a route from familiar photos such as your room, desk, commute, or favorite park. Place sticky-note style memory cards directly on each photo, then switch themes to reuse the same route for different topics such as exams, shopping lists, or presentation prep.

Features:

- Create ordered albums from place photos
- Organize notes by theme
- Place sticky notes on photos
- Review notes in route order
- Record simple review results
- View geotagged photos on a memory map

Mind Palace is intentionally lightweight. It helps you build a small walkable path in your head, then attach things you want to remember to that path.
