Privacy Policy

Memory Spots keeps your memory routes on your device.

Last updated: June 5, 2026

This policy explains how Memory Spots handles photos, locations, notes, and app preferences.

Information We Collect

Memory Spots does not use account registration, advertising, analytics services, tracking, or external server data transmission. Albums, themes, notes, added photos, photo locations, and app preferences created in the app are stored locally on the user's device.

Photos, Camera, and Location

The photo library is used to add user-selected photos to memory routes and to read location metadata when available. The camera is used to take place photos. Location is used to show the user's current position on the map and to associate captured or manually edited photos with a place.

Third Parties

Memory Spots does not sell, share, or transmit user data to third parties. The app does not include advertising SDKs or analytics SDKs.

Data Storage and Backup

Photo files are stored in app-managed local storage. The app excludes its stored photo directory from iCloud backup. Structured app data is stored locally using Apple's on-device persistence frameworks.

Data Deletion

You can delete albums, photos, themes, and notes in the app. Deleting Memory Spots from the device also deletes app data stored on that device.
